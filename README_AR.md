# طريقة نشر الصفحة مجانًا على GitHub Pages

## ما هذا الملف؟
هذا مجلد جاهز للنشر. الملف الأساسي هو:
- `index.html`

## النتيجة بعد النشر
سيكون عندك رابط عام يمكن وضعه في بايو إنستغرام، وأي شخص يفتحه سيستخدم الصفحة مباشرة بدون تسجيل دخول.

## أسهل طريقة

### 1) أنشئ حساب GitHub
إذا لم يكن لديك حساب، أنشئ حسابًا مجانيًا في GitHub.

### 2) أنشئ مستودعًا جديدًا
اسم مناسب مقترح:
- `ai-prompts-ar`
أو
- `prompt-guide-ar`

### 3) ارفع ملف `index.html`
بعد إنشاء المستودع:
- اختر إضافة ملف
- ارفع `index.html`
- احفظ التغييرات

### 4) فعّل GitHub Pages
من إعدادات المستودع:
- Settings
- Pages
- Source: Deploy from a branch
- Branch: main
- Folder: /root
ثم احفظ.

### 5) انتظر قليلًا
بعد دقائق سيظهر لك رابط مثل:
- `https://username.github.io/ai-prompts-ar/`

## ملاحظات مهمة
- الزائر لا يحتاج حساب GitHub.
- الصفحة ستكون عامة ومباشرة.
- يمكنك لاحقًا تعديل النصوص ورفع نسخة جديدة بسهولة.

## اقتراح اسم للرابط
- `abdullah-prompts`
- `learn-ai-prompts-ar`
- `ai-commands-ar`

## ماذا أضع في بايو إنستغرام؟
نص مقترح:
تعلم كتابة أوامر الذكاء الاصطناعي بأسلوب واضح ومجاني.

## ملاحظة تقنية
GitHub Pages يعمل مجانًا مع المواقع الثابتة مثل HTML/CSS/JavaScript.
